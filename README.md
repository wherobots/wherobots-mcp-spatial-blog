# wherobots-mcp-spatial-blog

Companion assets for a **Wherobots blog post** on MCP-driven spatial analysis with
[WherobotsDB](https://wherobots.com/), [Apache Sedona](https://sedona.apache.org/), and
[Overture Maps](https://overturemaps.org/) data.

> **This repository accompanies a blog post.** The notebooks and helper modules here are
> the runnable artifacts referenced in the article. Read the post for the full narrative
> and context.
>
> 📝 **Blog post:** _link coming soon_

## Contents

| Asset | Description |
|-------|-------------|
| _notebooks_ | Jupyter notebooks reproducing the analysis walked through in the post |
| _helper modules_ | Shared Plotly/visualization and summary helpers used by the notebooks |

_(Assets are added as the post is finalized.)_

## About the data

Examples are built on publicly available [Overture Maps Foundation](https://overturemaps.org/)
open data. Any social-media / event signals shown are **synthetic** and included purely to
demonstrate spatial-join and analysis workflows.

## Reproducing the notebooks

The notebooks run against a [WherobotsDB](https://wherobots.com/) environment and reference
catalogs available within that environment. They are provided primarily as an illustrative
companion to the blog post; running them end-to-end requires a Wherobots account and the
referenced data.
